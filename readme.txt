Easy Testimonials Plugin is a WordPress plugin to display your client review or testimonial in your WordPress website.

= Usage =

* Go to your Dashboard after installation and navigate to "Testimonial >> Settings" to configure the testimonial.

= Features =

  * Very easy installation
  * Shortcode system
  * Unlimited color variation by using jQuery color picker.
  * Responsive design
  * Flexible and easy to use
  * Lightweight JQuery


== Installation ==
1. Upload `Easy-testimonials-Slider-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>if(function_exists('wpetp_testimonial_shortcode')){wpetp_testimonial_loop();}</strong> in template code or  <strong>[WPETPTESTIMONIAL]</strong> in the post/page" to display testimonial.
